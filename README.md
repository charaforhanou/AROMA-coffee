# AROMA Coffee

A modern coffee house website with AI-powered features built during JobInTech training program.

---

## About

AROMA is an interactive coffee house website that combines elegant design with artificial intelligence. The project features an AI coffee detector using TensorFlow.js that can identify whether your coffee is hot or cold from an uploaded image.

---

## Features

- **AI Coffee Detector** - Machine learning model to classify coffee temperature
- **Geolocation** - Automatic visitor location detection
- **Shopping Cart** - Complete cart system with local storage
- **Smart Search** - Real-time menu filtering and sorting
- **Responsive Design** - Optimized for mobile, tablet, and desktop
- **Visit Counter** - Session-based tracking system

---

## Technologies

**Frontend**
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript ES6+

**AI & APIs**
- TensorFlow.js
- MobileNet (Pre-trained model)
- IPApi (Geolocation)

**Storage**
- localStorage
